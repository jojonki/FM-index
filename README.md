# BWT
The Burrows–Wheeler transform (also called block-sorting compression) 

## How to Use
```
python main.py abracadabra                                                              22:43:21  ☁  master ☂ ✭
Input: abracadabra
bwt: ['r', 'd', 'a', 'r', 'c', 'a', 'a', 'a', 'a', 'b', 'b']
Decoded: abracadabra
```
